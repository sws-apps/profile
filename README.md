# Kathreen Hervera — Portfolio

Personal portfolio of Kathreen Hervera — AI Developer, Product Manager, Systems Architect, and Licensed Professional Teacher. Founder of Scale With System.

Static site (HTML / CSS / vanilla JS), deployed via GitHub Pages.

- **Live:** https://sws-apps.github.io/profile
- **Edit content:** `index.html`
- **Styles:** `styles.css`
- **Behavior:** `script.js` (nav, scroll reveals, certificate lightbox)
- **Images:** `images/`

No build step. Open `index.html` locally or run `python3 -m http.server` in this folder to preview.
